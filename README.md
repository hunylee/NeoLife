# NeoLife AI

NeoLife AI investment landing page based on `NeoLife_AI_Investment Deck_2026.pdf`.

The page presents NeoLife AI as a personalized LifeCare and commerce platform that combines name, birth date and time, MBTI, blood type, apartment floor-plan feng shui analysis, and optional face analysis into one AI-powered experience.

## Included

- Investor-facing one-page website
- Deck-derived content rules in `AGENTS.md`
- Generated hero image asset in `public/assets/neolife-ai-hero.png`
- Dependency-free static HTML, CSS, and JavaScript

## View Locally

Open `index.html` directly in a browser, or run a local static server:

```bash
python3 -m http.server 4173
```

Then visit:

```text
http://localhost:4173
```
