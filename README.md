## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) v18+  *(or your runtime of choice)*
- [Git](https://git-scm.com/)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/NeoLife.git

# 2. Navigate into the project
cd NeoLife

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```
